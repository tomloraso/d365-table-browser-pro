I built a free browser extension for D365 F&O consultants — and it does more than just open the Table Browser.

Most D365 table browser tools do one thing: redirect you to SysTableBrowser. That's fine until your org disables it on production, or you need to actually query data, or you're juggling three environments and can't remember which URL is which.

I wanted something better. So I built D365 F&O Table & OData Browser Pro.

Instant table search — 15,800+ D365 tables searchable as you type. No metadata loading, no waiting. Results appear instantly because the full table list is bundled in the extension.

Query data without SysTableBrowser — The built-in OData Query Builder lets you pick entities, select fields, add filters, and execute queries directly. Results show in a sortable data grid you can export to CSV, paste into Excel as TSV, or copy as JSON. No special permissions required — it uses OData, which is available on every environment.

Cross-company queries in one click — This is the big one. Need to see inventory across all legal entities? Customer balances across every company? With SysTableBrowser you're stuck in one company at a time. The OData Query Builder has a cross-company toggle — flip it on and query across every legal entity in a single request. No more switching companies and re-running the same query ten times.

Multi-environment support — Name your environments (Dev, UAT, Prod), assign colors, and switch between them in one click. The extension auto-detects which environment you're on and which company you're in.

Respects your D365 security — The extension uses your existing browser session and D365 enforces all permissions server-side. You can only see data your security role allows. No credentials stored, no elevated access.

Side panel mode — Pin it alongside D365 so you can browse tables and query data without losing your place.

Works on Chrome, Edge, and Firefox.
