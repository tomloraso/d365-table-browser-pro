I built a free browser extension for anyone working in D365 F&O.

It does two things: lets you search and open any of 15,800+ D365 tables in one click, and gives you a visual OData Query Builder so you don't have to build URLs by hand.

A few things that make it useful day-to-day:

- Cross-company queries in one click. Toggle it on and pull data across all legal entities in a single request. No more switching companies and re-running the same query.

- Results show in a sortable data grid. Export to CSV, copy as TSV straight into Excel, or grab the JSON.

- Multi-environment support. Set up your Dev, UAT, and Prod environments with color-coded badges, switch between them from the dropdown.

- Side panel mode. Pin it alongside D365 so you can browse tables and query data without leaving your page.

It uses your existing D365 browser session — no credentials stored, no elevated access. D365 enforces all security server-side, so you can only see what your role allows.
