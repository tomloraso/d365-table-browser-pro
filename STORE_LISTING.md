# Chrome Web Store / Edge Add-ons Listing

## Extension Name
D365 F&O Table & OData Browser Pro

## Short Description (132 char max for Chrome Web Store)
Browse D365 F&O tables instantly, query OData entities, and export data. Multi-environment support for Dynamics 365 consultants.

## Detailed Description

The most powerful table browser extension for Dynamics 365 Finance & Operations.

Instantly search 15,800+ D365 F&O tables by name, open the Table Browser with one click, and build OData queries visually - all from your browser toolbar or side panel.

FEATURES:

- Instant Table Search: Search 15,800+ D365 tables as you type. No metadata loading required - results appear instantly.

- Multi-Environment Support: Manage multiple D365 environments (Dev, UAT, Prod) with color-coded badges. Switch between environments in one click.

- Visual OData Query Builder: Select entities, pick fields, add filters, and configure options ($top, $orderby, cross-company). See the OData URL in real-time.

- Data Preview & Export: Execute queries and view results in a sortable data grid. Export to CSV, copy as TSV (paste into Excel), or copy as JSON.

- OData Entity Explorer: Browse all OData entities from your environment's $metadata. View field names, types, and descriptions.

- Favorites & History: Star frequently used tables. Your recent history is saved across sessions.

- Side Panel Mode: Pin the extension as a side panel for an expanded workspace while you browse D365.

- Security First: Uses your existing D365 browser session - no passwords or API keys stored. Read-only queries only.

BUILT FOR D365 CONSULTANTS:

Whether you're a functional consultant checking table data, a developer debugging integrations, or a solution architect reviewing entity structures - this extension saves you time every day.

Works with:
- Dynamics 365 Finance
- Dynamics 365 Supply Chain Management
- Dynamics 365 Commerce
- All D365 F&O cloud environments (sandbox, UAT, production)

Supports Chrome, Edge, and Firefox.

# How to Publish

## Chrome Web Store
1. Go to https://chrome.google.com/webstore/devconsole
2. Pay one-time $5 developer fee (if not already registered)
3. Click "New Item" > upload the `.output/chrome-mv3` folder as a ZIP
4. Fill in the listing details above
5. Add screenshots (1280x800 or 640x400)
6. Submit for review (typically 1-3 business days)

## Microsoft Edge Add-ons
1. Go to https://partner.microsoft.com/en-us/dashboard/microsoftedge
2. Sign in with your Microsoft account (free)
3. Click "New extension" > upload the same Chrome ZIP (Edge accepts MV3)
4. Fill in listing details
5. Submit for review (typically 1-2 business days)

## Firefox Add-ons (AMO)
1. Run: `npx wxt build -b firefox`
2. Go to https://addons.mozilla.org/en-US/developers/
3. Click "Submit a New Add-on" > upload the `.output/firefox-mv2` folder as a ZIP
4. Fill in listing details
5. Submit for review
